# refoliate

A suite of tools for semantic code indexing, embedding visualization and agentic
workflow management.

## Aerie

A node graph workbench for creating agentic workflows.

## Embasee

Uses UMAP dimensionality reduction to visualize semantic embeddings.

## embcp-server

An example MCP server for semantic searches on code snippets.

## emberlain

A tool to perform semantic indexing on a code repository.

Runtime extensibility via tree-sitter queries and wasm encoded grammar files.
